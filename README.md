# RDR2 Wildlife API

ASP.NET Core 8 REST API for cataloging Red Dead Redemption 2 
wildlife, built with Entity Framework Core.

## Data Model
- Animals categorized by size (Small → Legendary)
- Many-to-many relationships: Animals ↔ Weapons, Animals ↔ Locations  
- User authentication with role-based access (User/Admin)
- Weapon types: Bullets, Arrows, Shotgun Shells

## Tech Stack
ASP.NET Core 8 · Entity Framework Core · Swagger · SQL Server
